## C -> Zig rewrite

I'm rewriting this codebase from C to [Zig](https://ziglang.org). If you're reading this notice, it's in an intermediate state.

## quickstart demo

```
~ $ git clone https://github.com/rossry/luminary-zig.git
~ $ cd luminary-zig
~/luminary-zig $ git submodule update --init
~/luminary-zig $ scripts/install.sh
~/luminary-zig $ scripts/generate.sh
```
